## 📘 Introduction

This project analyzes Airbnb listing data and transforms it into an interactive web dashboard.

The dataset was cleaned, explored, and visualized using Python before being integrated into a Streamlit application.

The dashboard helps users explore accommodation prices, room types, ratings, availability, and geographical patterns across different destinations.
